# Лабораторная работа №13-14
Коллекции, обобщения и функциональный стиль в Kotlin
## Описание
Данная лабораторная работа посвящена изучению продвинутых возможностей языка Kotlin,которые активно используются при разработке Android-приложений.
В рамках работы рассматриваются:
- обобщённые типы (Generics);
- коллекции Kotlin;
- функции высшего порядка;
- extension-функции;
  Все примеры ориентированы на практическое применение и подготовку к разработкемобильных приложений.

## Изученное в этой лабораторной

### Generics
это способ писать универсальный код, который работает с любыми типами данных, а конкретный тип передаётся при использовании, а не при написании.

```class Box<T>(val value: T)```
```val intBox = Box<Int>(10)```
```val stringBox = Box<String>("Hello")```

### Enum class
это класс, который хранит фиксированный набор констант
```enum class ResourceType {```
```   MINERALS,```
```  GAS,```
```  ENERGY```
```}```

### data class
это класс для хранения данных,
в котором Kotlin автоматически генерирует служебный код.
```data class User(val name: String, val age: Int)```
## singleton-объекта
это шаблон проектирования, при котором существует только один экземпляр класса на всю программу.
```object Logger {
    fun log(message: String) {
        println(message)
    }
}
